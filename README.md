Architecture Decisions
Why multi-tenant over separate databases?
Multi-tenancy with Supabase RLS keeps one deployment, one schema, and one codebase — while enforcing complete data isolation through database policy rather than application logic. The constraint lives at the database level where it cannot be bypassed.
Why React Context over Redux?
React Context with useMemo derivations provides the same single-source-of-truth pattern with less overhead, better TypeScript integration, and no additional dependencies. One fetch. Consistent everywhere.
Why Anthropic API for contradiction detection?
The competitive advantage is not the model — it is the prompt engineering layer and the product context around it: knowing which fields to compare, what severity means in an immigration context, and how to render output as actionable UI.
